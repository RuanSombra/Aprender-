# Aprender

É uma aplicativo que visa incentivar os estudos por meio de um quiz.

# As Tecnologias Usadas:

- Flutter;
- Dart;
- Local Storage (Armazenamento na memória).

# Quando foi desenvolvido?

Foi desenvolvido como treinamento para WordlSkills 2025 Nacional. Esse projeto de software é um dos roteiros seguidos para fim de conhecimento, desenvolvimento e prática.

# Como rodar?

Preciso que você utilize a branch = master (local real do projeto).
